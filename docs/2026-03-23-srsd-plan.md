# 服务注册与发现（SRSD）原型生成计划

> 项目名称：服务注册与发现 — Panther 平台
> 生成日期：2026-03-23
> PRD 来源：`CurrentSituation/SRSD-1.md`

---

## 1. 需求理解汇总

**一句话描述**：管理命名空间、服务、访问申请与审核，以及个人授权凭证的微服务注册与发现模块。

**核心用户场景**（按优先级）：
1. 查看/创建/管理命名空间
2. 查看/创建/管理服务（含实例管理与监控）
3. 提交和跟踪服务访问申请
4. 审核他人的访问申请（管理员/服务持有人）
5. 查看个人信息与已授权服务凭证

**功能边界**：
- 做：命名空间、服务、实例、申请、审核、个人信息的增删改查与展示
- 不做：真实服务注册逻辑与后端对接；监控用 mock 数据模拟图表

## 2. 页面清单与关联

| 文件名 | 页面类型 | 职责 |
|--------|----------|------|
| `namespace-list.html` | 列表管理页 | 命名空间列表，含新建/编辑弹窗 |
| `service-list.html` | 列表管理页 | 服务列表，含新建弹窗，点击服务名跳转详情 |
| `service-detail.html` | 详情页（4 Tab） | 服务信息 / 实例列表 / 已授权服务端 / 服务监控 |
| `apply-list.html` | 列表管理页 | 申请列表，含新建申请弹窗，支持重新申请 |
| `review-list.html` | 列表管理页 | 审核列表，支持通过/拒绝操作 |
| `profile.html` | 配置页（2 Tab） | 个人信息 / 已授权服务端凭证 |

**跳转关系**：
- `namespace-list.html` ← 侧边栏默认落地页
- `service-list.html` → `service-detail.html?id=xxx`（点击服务名）
- `apply-list.html` → 重新申请弹窗（状态为"未通过"时）

**侧边栏菜单**：
```
服务注册与发现
├── 🗂️ 命名空间 → namespace-list.html
├── 🔧 服务 → service-list.html
├── 📋 申请管理 → apply-list.html
├── ✅ 审核管理 → review-list.html
└── 👤 我的 → profile.html
```

## 3. 每页生成要点

### namespace-list.html
- 布局：搜索栏（空间类型下拉 + 业务线下拉 + 关键词输入）+ 数据表格 + 分页 + 新建/编辑弹窗
- 弹窗字段：空间名称（格式校验）、空间类型（下拉）、空间描述
- 种子数据：7 条（image-delete、domeos-pub、schuro-public、sohucs、cloud-native、panther-backend、panther-web）

### service-list.html
- 布局：搜索栏（命名空间下拉 + 业务线下拉 + 关键词输入）+ 数据表格 + 分页 + 新建服务弹窗
- 弹窗字段：命名空间（下拉）、服务名（格式校验）、实例地址类型（下拉）、协议类型（下拉）、鉴权（开关）、网关token、日志收集（开关）、服务描述
- 种子数据：10 条（vip、schuro-cloud、jenkins、argo 等）

### service-detail.html
- 布局：面包屑 + 返回按钮 + 4 Tab 切换
  - **服务信息**：描述列表 + 编辑服务按钮（弹窗）
  - **实例列表**：实例表格（含状态标签）+ 新增/编辑/删除弹窗
  - **已授权服务端**：公共凭证（User/Password 显隐+复制）+ 私有凭证列表
  - **服务监控**：实例筛选 + 时间范围切换 + 折线图 + 延迟指标卡片
- 特殊处理：引入 ECharts CDN；URL 参数 `?id=` 读取 sessionStorage 数据

### apply-list.html
- 布局：搜索栏（状态下拉 + 关键词）+ 表格（含彩色状态标签）+ 分页 + 新建申请弹窗
- 状态标签：已通过（`tag-success`）/ 未通过（`tag-error`）/ 审核中（`tag-processing`）
- 操作列：「查看」始终显示；「重新申请」仅未通过时显示
- 种子数据：4 条（ID 35/37/40/41）

### review-list.html
- 布局：同申请列表，含审核操作弹窗（通过/拒绝）
- 种子数据：3 条审核中的数据

### profile.html
- 布局：2 Tab（个人信息 / 已授权服务端）
  - 个人信息：只读描述列表
  - 已授权服务端：公共凭证（显隐+复制）+ 私有凭证搜索表格（暂无数据）
