# 我的 交互规范

> 对应页面：`profile.html` · 页面类型：配置页

---

## 1. 页面概览
- 页面入口：`profile.html`
- 页面类型：配置页（多 Tab）
- 核心功能：展示当前登录用户的个人信息；以及与服务访问相关的授权凭证（公共凭证 + 私有凭证列表）

## 3. 组件清单

| 组件 | 职责 | 备注 |
|------|------|------|
| Tabs 导航 | 切换个人信息 / 已授权服务端 | |
| 个人信息描述列表 | 展示用户基本信息 | 只读 |
| 公共凭证区域 | 展示全局 User/Password | 含显隐密码、复制按钮 |
| 私有凭证表格 | 展示已授权服务的专用凭证 | 含搜索，初始暂无数据 |

## 4. 数据模型

### 个人信息

| 字段 | 类型 | 说明 |
|------|------|------|
| 姓名 | string | 只读，当前用户姓名 |
| 邮箱 | string | 只读，企业邮箱 |
| 部门 | string | 只读，所属部门 |
| 用户名 | string | 只读，系统用户名（即公共凭证 User） |

### 公共凭证

| 字段 | 类型 | 说明 |
|------|------|------|
| User | string | 与用户名相同，`qinghualv` |
| Password | string | 全局访问密码，默认遮掩显示 |

### 私有凭证列表

| 字段 | 类型 | 说明 |
|------|------|------|
| svcName | string | 被授权的服务名称 |
| proxy | string | 服务代理地址 |
| ns | string | 所属命名空间 |
| user | string | 该服务专用用户名 |
| pwd | string | 该服务专用密码，默认遮掩 |

## 5. 交互流程

### 5.1 Tab 切换
1. 默认显示"个人信息" Tab
2. 点击"已授权服务端" Tab → 切换到凭证区域，初始化私有凭证表格

### 5.2 个人信息展示
1. 页面加载后，个人信息以描述列表形式展示（只读，无任何操作按钮）

### 5.3 公共凭证操作
1. User 字段旁显示「复制」按钮 → 点击后复制 User 文本到剪贴板 → Toast "已复制到剪贴板"
2. Password 字段默认显示 ••••••••••••••••（遮掩）
3. 点击「显示」→ Password 明文显示，按钮变为"隐藏"
4. 点击「隐藏」→ Password 重新遮掩
5. 点击 Password 旁「复制」→ 复制真实密码到剪贴板 → Toast "已复制到剪贴板"

### 5.4 私有凭证搜索
1. 在搜索框输入关键词（服务名称/所属空间），按回车 → 凭证列表按关键词过滤
2. 清空搜索框 → 列表恢复全量数据
3. 当前示例数据为空，显示"暂无数据"
