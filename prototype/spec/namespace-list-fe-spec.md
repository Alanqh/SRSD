# 命名空间列表 交互规范

> 对应页面：`namespace-list.html` · 页面类型：列表管理页

---

## 1. 页面概览
- 页面入口：`namespace-list.html`（侧边栏默认落地页）
- 页面类型：列表管理页
- 核心功能：展示当前用户可见的命名空间，支持按类型/业务线/关键词筛选，支持新建、编辑、删除空间

## 2. 布局结构

```
┌────────────────────────────────────────────────────────┐
│ 面包屑：服务注册与发现 / 空间列表                        │
│ 页面标题：空间列表                          [+ 新建]    │
├────────────────────────────────────────────────────────┤
│ 空间类型[▼]  业务线[▼]  关键词[______]  [查询][重置]    │
├────────────────────────────────────────────────────────┤
│ 空间名称  空间类型  描述  服务数  持有人  业务线  操作   │
│ ──────────────────────────────────────────────────     │
│ image-delete  私有  删除图片  1  王立  北京研发部  编辑 删除 │
│ domeos-pub    公共  DomeOS…  1  刘振  --         编辑 删除 │
│ ...                                                    │
├────────────────────────────────────────────────────────┤
│ 共 N 条                              ‹ 1  2 ›          │
└────────────────────────────────────────────────────────┘
```

> 新建/编辑弹窗为浮层，删除确认框为浮层

## 3. 组件清单

| 组件 | 职责 | 备注 |
|------|------|------|
| 搜索栏 | 按空间类型/业务线/关键词筛选 | 回车触发查询 |
| 数据表格 | 展示命名空间列表 | |
| 操作列 | 每行的编辑/删除按钮 | |
| 分页组件 | 翻页控制 | 每页 10 条 |
| 新建/编辑弹窗 | 表单提交 | 新建/编辑共用，编辑时空间名称不可修改 |
| 删除确认框 | 二次确认 | 显示空间名称 |
| Toast 消息 | 操作结果反馈 | |

## 4. 数据模型

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 唯一标识，只读，由 `genId()` 生成 |
| name | string | 是 | 空间名称，创建后不可修改；以小写字母开头，可包含小写字母、数字和连字符，不能以连字符结尾 |
| type | `'public' \| 'private'` | 是 | 空间类型：公共空间 / 私有空间 |
| desc | string | 否 | 空间描述 |
| serviceCount | number | 是 | 该空间下的服务数量，只读 |
| owner | string | 是 | 空间持有人，只读 |
| bizline | string | 否 | 所属业务线 |
| createdAt | string | 是 | 创建时间，ISO 8601，只读 |
| updatedAt | string | 是 | 最后更新时间，ISO 8601，只读 |

## 5. 交互流程

### 5.1 搜索筛选
1. 用户选择空间类型、业务线，或在关键词框输入文字
2. 点击「查询」或在关键词框按回车 → 表格按条件过滤 → 分页重置为第 1 页
3. 点击「重置」→ 所有搜索条件清空 → 表格恢复全量数据 → 分页重置为第 1 页

### 5.2 新建空间
1. 点击「+ 新建」→ 弹窗打开，标题"新建空间"，所有字段清空，空间名称可编辑
2. 用户填写空间名称（必填）、空间类型（必填）、空间描述（选填）
3. 点击「确定」→ 触发字段校验
4. 校验失败 → 对应字段下方显示错误提示，弹窗不关闭
5. 校验通过且名称未重复 → 新空间出现在列表第一行 → 弹窗关闭 → Toast 显示"创建成功"
6. 空间名称已存在 → 显示"空间名称已存在"，弹窗不关闭

### 5.3 编辑空间
1. 点击某行「编辑」→ 弹窗打开，标题"编辑空间"，回填该行数据；空间名称置灰不可修改
2. 用户修改空间类型或描述
3. 点击「确定」→ 触发字段校验
4. 校验通过 → 列表对应行数据更新 → 弹窗关闭 → Toast 显示"编辑成功"
5. 点击「取消」或弹窗遮罩 → 弹窗关闭，列表不变

### 5.4 删除空间
1. 点击某行「删除」→ 弹出确认框，文案："确认删除命名空间「[name]」？删除后无法恢复。"
2. 点「确定」→ 该空间从列表移除 → 确认框关闭 → Toast 显示"删除成功"
3. 点「取消」→ 确认框关闭，列表不变

## 6. 前端校验规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| name | 必填；以小写字母开头，可含小写字母、数字、连字符，不能以连字符结尾；不能与现有名称重复 | "空间名称不能为空" / "以小写字母开头，可包含小写字母、数字和连字符，不能以连字符结尾" / "空间名称已存在" |
| type | 必填，从下拉选项中选择 | "请选择空间类型" |

## 7. 验收标准

- [ ] 操作：选择空间类型"私有空间"后点击「查询」 → 预期结果：表格只显示私有空间
- [ ] 操作：输入关键词点击「查询」 → 预期结果：表格只显示名称/描述/持有人包含关键词的记录
- [ ] 操作：点击「重置」 → 预期结果：筛选条件清空，表格恢复全量数据
- [ ] 操作：新建时空间名称为空点击「确定」 → 预期结果：显示"空间名称不能为空"，弹窗不关闭
- [ ] 操作：新建时输入已存在的名称 → 预期结果：显示"空间名称已存在"，弹窗不关闭
- [ ] 操作：成功新建空间 → 预期结果：新空间出现在列表第一行，Toast 显示"创建成功"
- [ ] 操作：点击某行「编辑」 → 预期结果：弹窗打开，回填数据，空间名称不可编辑
- [ ] 操作：编辑成功 → 预期结果：列表数据更新，Toast 显示"编辑成功"
- [ ] 操作：点击「删除」并确认 → 预期结果：该空间从列表消失，Toast 显示"删除成功"
- [ ] 操作：点击「删除」后取消 → 预期结果：列表不变
