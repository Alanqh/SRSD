# 服务详情 交互规范

> 对应页面：`service-detail.html` · 页面类型：详情页

---

## 1. 页面概览
- 页面入口：`service-detail.html?id={id}`
- 页面类型：详情页（多 Tab）
- 核心功能：展示服务完整信息，分四个 Tab：服务信息、实例列表、已授权服务端、服务监控

## 2. 布局结构

```
┌────────────────────────────────────────────────────────────┐
│ 面包屑：服务注册与发现 / 服务列表 / {服务名}                  │
│ 页面标题：{服务名}                                [返回]     │
├────────────────────────────────────────────────────────────┤
│ [服务信息] [实例列表] [已授权服务端] [服务监控]               │
├────────────────────────────────────────────────────────────┤
│                                              [编辑服务]     │
│ 命名空间     changjie-test   业务线     北京研发中心          │
│ 服务名称     test-health-...  协议类型  http                 │
│ 鉴权         已关闭           代理地址  …                    │
│ ...                                                        │
└────────────────────────────────────────────────────────────┘
```

> id 不存在时显示"服务不存在"结果态

## 3. 组件清单

| 组件 | 职责 | 备注 |
|------|------|------|
| Tabs 导航 | 切换四个功能区 | 切换时不刷新页面 |
| 服务信息描述列表 | 展示服务详细字段 | 只读 |
| 编辑服务弹窗 | 修改鉴权/日志/token/描述 | 服务名/命名空间不可改 |
| 实例表格 | 展示实例列表 | 含健康状态标签 |
| 新增/编辑实例弹窗 | 管理实例 | |
| 公共凭证区 | 展示 User/Password | 含显隐密码、复制功能 |
| 私有凭证表格 | 展示已授权的私有服务凭证 | 含搜索 |
| 服务监控图表 | 折线图展示请求速率 | 使用 ECharts，含时间范围切换 |
| 延迟指标卡片 | 展示三项延迟指标 | 静态展示 |
| 结果态 | id 不存在时展示 | 含"返回服务列表"按钮 |

## 4. 数据模型

### 服务数据
从 `srsd_services` sessionStorage 中按 id 读取（详见 service-list-fe-spec.md）

### 实例数据（存储 key：`srsd_instances_{serviceId}`）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 实例唯一 ID，格式：`priv-{ns}-{svcName}-kdcuqi4p-{随机串}` |
| svcId | string | 是 | 所属服务 ID |
| addr | string | 是 | 实例 IP 地址 |
| port | number | 是 | 监听端口，范围 1-65535 |
| dc | `'办公' \| '生产-北京' \| '生产-上海'` | 是 | 所在机房 |
| status | `'healthy' \| 'unhealthy'` | 是 | 健康状态 |
| createdAt | string | 是 | ISO 8601，只读 |

## 5. 交互流程

### 5.1 页面初始化
1. 读取 URL 参数 `?id=`，从 sessionStorage 中查找对应服务记录
2. 找到 → 显示详情区域，渲染服务信息 Tab
3. 找不到 → 显示"服务不存在"结果态，提供「返回服务列表」链接

### 5.2 Tab 切换
1. 点击 Tab 导航项 → 对应内容面板显示，其他面板隐藏
2. 首次切换到"实例列表" → 渲染实例表格
3. 首次切换到"已授权服务端" → 渲染凭证表格（初始暂无数据）
4. 首次切换到"服务监控" → 初始化 ECharts 图表

### 5.3 编辑服务信息
1. 点击「编辑服务」→ 弹窗打开，回填鉴权/日志/token/描述；服务名和命名空间置灰不可修改
2. 修改后点「确定」→ 服务信息更新保存到 sessionStorage → 弹窗关闭 → Toast "保存成功"

### 5.4 实例管理
1. 点击「+ 新增实例」→ 弹窗打开，字段清空
2. 填写实例地址、端口（必填），选择机房，点「确定」→ 触发校验
3. 校验通过 → 新实例添加到列表 → Toast "新增成功"
4. 点击某实例「编辑」→ 弹窗打开，回填该实例数据
5. 修改后确认 → 实例数据更新 → Toast "编辑成功"
6. 点击某实例「删除」→ 弹出确认框 → 确认后从列表移除 → Toast "删除成功"

### 5.5 凭证复制与显隐
1. 点击 User 旁的「复制」→ 复制 User 文本到剪贴板 → Toast "已复制到剪贴板"
2. 点击 Password 旁的「显示」→ 密码明文显示，按钮变为"隐藏"
3. 点击「隐藏」→ 密码重新遮掩为 ••••••••••••••••
4. 点击 Password 旁的「复制」→ 复制真实密码到剪贴板 → Toast "已复制到剪贴板"

### 5.6 服务监控
1. 页面加载（切换到监控 Tab 时）→ 初始化折线图，默认"Last 30 minutes"
2. 切换时间范围下拉 → 图表数据刷新（X 轴时间标签和数据均更新）
3. 窗口 resize → 图表自动 resize

## 6. 前端校验规则

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 实例地址 | 必填，不为空 | "实例地址不能为空" |
| 端口 | 必填，整数，范围 1-65535 | "请输入有效端口（1-65535）" |
