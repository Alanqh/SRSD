Token 管理 🔐 Master 及以上

ℹ️ Token 是 Consul ACL Token 的业务抽象，由系统自动生成，无需手动填写。一个 Token 可关联同一命名空间下的多个真实服务（虚拟服务不支持关联）。
如何在 DomeOS 中使用 Token：
步骤一：在本页面创建 Token，点击列表中「复制」按钮可复制完整 Token。
步骤二：在 DomeOS 平台进入目标部署的配置页面，找到「服务注册」相关设置。（待完善：具体操作路径补充中…）
步骤三：将 Token 值填入 DomeOS 对应字段，并保存配置，服务实例将自动使用该 Token 完成 Consul 注册鉴权。（待完善：字段名称及截图补充中…）
