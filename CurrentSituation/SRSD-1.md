## 8. 服务 - 新建服务

**截图文件：** `service/新建服务.png`

### 交互方式
弹窗（Modal）形式，点击服务列表页「新建」按钮触发。

### 表单字段

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| 命名空间 | 是 | 下拉选择 | 选择服务所属的命名空间 |
| 服务名 | 是 | 文本输入 | 以小写字母开头，可包含小写字母、数字和连字符，不能以连字符结尾 |
| 实例地址类型 | 是 | 下拉选择 | 指定实例地址的类型 |
| 协议类型 | 是 | 下拉选择 | 服务通信协议 |
| 鉴权 | 是 | 开关 | 开启后访问服务代理地址时需要携带有效的身份凭证才能访问 |
| 网关token | 否 | 文本输入 | 设置后 HTTP 请求经过代理地址时，Header 中会自动增加 `Authorization: $字段值` 头部 |
| 日志收集 | 是 | 开关 | 是否收集该服务的访问日志 |
| 服务描述 | 否 | 多行文本 | 建议对服务的主要功能、适用范围进行描述，可附加接口文档地址 |

### 操作按钮
- 取消
- 确定

## 12. 服务详情 - 服务监控

**截图文件：** `service/服务详情-服务监控.png`

### 页面功能
- 实例筛选：Instance - All（可按实例过滤）
- 时间范围：Last 30 minutes，支持自定义时间范围
- 自动刷新：5s

### 监控指标

| 指标 | 类型 | 说明 |
|------|------|------|
| HTTP 总请求速率 | 折线图 | Requests/second，展示整体请求量趋势 |
| 各 HTTP 状态码的请求速率 | 折线图 | 按状态码（如 HTTP Status 200、HTTP Status 404）分色展示 |
| HTTP 请求延迟 | 指标卡片 | 端到端请求响应时间 |
| APISIX 内部处理延迟 | 指标卡片 | 网关自身处理耗时 |
| 上游服务响应延迟 | 指标卡片 | 后端实例响应耗时 |
